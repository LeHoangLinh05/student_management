
### front
```
npm install

```
### back
```
cd backend
npm install
npm run dev
``` 
## Docker
Tạo file .env theo mẫu .env.example tại:
```
D:\Blockchain\student_management\
D:\Blockchain\student_management\backend\
D:\Blockchain\student_management\backend\blockchain\
```

### Build image lần đầu:
```
docker compose build
```
### Hoặc gộp cả build + run:
```
docker compose up --build
```
### Run : 
```
http://localhost:5173/
```
